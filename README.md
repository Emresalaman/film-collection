# 🎬 Film Koleksiyonu

Kişisel film koleksiyonunu yönetebileceğin bir web uygulaması.

## Özellikler

- Film ekleme, listeleme, güncelleme ve silme (CRUD)
- Türe ve izlenme durumuna göre filtreleme
- Arama
- Toplam film, izlenen ve ortalama puan istatistikleri

## Teknolojiler

- React 18
- Vite
- Tailwind CSS

## Kurulum

```bash
npm install
npm run dev
```
